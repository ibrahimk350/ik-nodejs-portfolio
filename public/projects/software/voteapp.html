Vote Web App


Project Description:
This project was built in JSP and Servlet using Netbeans. It is a simple web app which demonstrates how one can create Java based web application. It is also connected to database in order to validate the correct voter who logs in to the site. For its front, it uses JSP while for the back where requests are posted it uses Java Servlet.


Project Technologies:
  • Java
  • Java Server Pages (JSP)
  • Servlets
  • Database
  • NetBeans
