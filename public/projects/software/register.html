Course Registration System


Project Description:
This program is written in Java which demonstrates the school registeration system on a small scale. This program shows the registered students for specific courses. Moreover, the system administrator can see register students as well as registered courses. They can also register new students and assign them courses as well as register courses. It is a dynamic course registeration system which is connected to MySQL database. It uses WindowsBuider for the GUI and can be shipped as JAR or an executable file for windows.


Project Technologies:
  • Java
  • MySQL Database
  • Windows Builder
  • Eclipse
