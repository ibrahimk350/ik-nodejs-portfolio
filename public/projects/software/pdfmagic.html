PDF Magic


Project Description:
This project was built in Java with the help of WindowsBuilder. This project allows a company to convert powerpoint files to pdf files for each and every employee. Once the file is converted, it then watermarks the entire pdf file and secures the file. It allows managers to track the company's pdf files and eliminte the possibility of the file being stolen. This project uses database to track the files and iTextPdf library along with Open Office convertor in order to convert big files from PowerPoint to PDF.


Project Technologies:
  • Java
  • iTextPdf
  • OpenOffice
  • Database
  • JavaScript
