GradeBook - C#


Project Description:
This is a simple program written in C#. This is a simple gradebook where the professor provides course code and updates student marks along with student information. It shows them the lowest mark achieved in class, the highest mark achieved in class as well as the class average marks. This was a fun project which I completed in about two days for my own benefit.


Project Technologies:
  • C#
  • Visual Studio 2015
  • Database
