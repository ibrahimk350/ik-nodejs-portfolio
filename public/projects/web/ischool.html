iSchool Website


Project Description:
Personal Project built in ASP.Net demonstrating a school system on a small scale. The website is connected to database. Students can be registered for courses as well as have their student information updated. Teachers can view their courses along with students that are enrolled in a specific course. System administartos can add teachers, students, courses as well as remove them. They can also edit website content. This project is almost like an admin control panel which only the staff is allowed to access.


Project Technologies:
  • Developed in ASP.NET
  • Database
  • Visual Studio 2015
  • SQL Server Managment Studio
  • JavaScript
