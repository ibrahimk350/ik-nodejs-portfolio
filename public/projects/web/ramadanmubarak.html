RamadanMubarak Website


Project Description:
RamadanMubarak.ca is a custom made PHP framework to serve the needs of the Muslim community all around the World. The website has been hand crafted to perfection with the main noticable feature on the front page being a prayer time calculation based on the user's local time. The background changes real time in order to reflect the user's local time. With Facebook and Twitter's integrated authentication, users can challenge one another in multiple quizzes and post the result to their social network profile. More features include; nearest mosque locator, custom calendar, custom articles, and many more.


Project Technologies:
