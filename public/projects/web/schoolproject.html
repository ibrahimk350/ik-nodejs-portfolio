School Sponsored Project Website


Project Description:
This was a school project developed for a professor at Centennial College. This is a dynamic website which is connected to a database. Visitors can only browse the contents of the website. Authorized users on the other hand can login to make changes to the website. It uses a custom made content slider along with two front widgets which were the requirements of the project. It also feature an announcement section which is displayed on the front page of the website. Feel free to have a look at the screen shots in order to get a better understanding of the project.


Project Technologies:
  • Developed in ASP.NET
  • Micrsoft Visual Studio 2015
  • SQL Server Management Studio
  • Database
  • JavaScript
