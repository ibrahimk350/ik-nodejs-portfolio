R.M.A(Ramadan Mubarak Assistant) Android App


Project Description:
R.M.A is a companion application developed for RamadanMubarak.ca which is a custom PHP Framework. The application has online and offline mode. Offline mode gives them access to all the essential features but locally. The onlnie mode gives them access to all the features that the application has. Whether the user is connected to internet or not, it always ensure that the application is still accessible.


Project Technologies:
  • Java
  • PHP
  • Database
  • Android Studio
