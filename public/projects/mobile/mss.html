Muslim Students Society Android App


Project Description:
This application was a community messaging system developed in the first year of my software degree. This application had a real time chat feature, along with forums and suggest board. It also featured application wide announcements. Moreover, it had a compass and authentication system that could allow users to register and login in order to view their own personalized feed.


Project Technologies:
  • Java
  • Database
  • PHP
  • Scringo
  • Google Cloud Messaging
  • Forums
  • Suggestions Board

Project Status:
  • Discontinued
