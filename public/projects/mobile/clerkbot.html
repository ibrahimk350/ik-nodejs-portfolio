ClerkBot Android App


Project Description:
This project was for UCaaS(Unified Communication as a Service) Maker Challenge. Me and my team worked with Cisco Spark API to create an android chat application in order to understand the underlying technologies. Using our custom made application, we integrated that with our Cisco Bot (ClerkBot) and Google Calendar. The bot could create meetings for you, take daily attendance, assign action items, send email reminders to your team mates. It is the initial marking of AI entering the field of Project Management.


Project Technologies:
  • Cisco Spark API
  • Java
  • Google OAuth (To authenticate users)
  • Voice API
  • PHP
  • Android Studio

Project Status:
